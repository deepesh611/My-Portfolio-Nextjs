## Overview

Video Summary Generator is a **multimodal deep learning application** that automatically generates intelligent summaries from video content by processing **visual and audio content together** at synchronized time intervals.  

Unlike traditional approaches that process modalities separately, this system uses **time-synchronized extraction** where each moment in the video is analyzed both visually (using BLIP-2) and aurally (using Whisper), ensuring perfect alignment between what is seen and what is heard. This synchronized approach creates richer, more accurate video summaries.

The project demonstrates advanced multimodal AI techniques, serverless GPU deployment, and production-ready ML pipeline design.

## Key Features

- **Synchronized Multimodal Processing** – Extracts (frame + audio) pairs at regular time intervals ensuring perfect audio-visual alignment
- **Advanced Visual Understanding** – BLIP-2 (2.7B parameters) generates natural language descriptions of video frames
- **Speech Recognition** – OpenAI Whisper transcribes audio segments with multiple model size options (tiny to large)
- **Intelligent Moment Selection** – Automatically identifies ~20 key moments from the video timeline
- **LLM-Based Narrative Generation** – Optional OpenRouter API integration for coherent summary narratives
- **Cloud Deployment** – Backend on Modal (serverless GPU), frontend on Streamlit Cloud
- **Production-Ready API** – FastAPI backend with proper error handling and file validation

## Architecture

The system uses a **synchronized multimodal pipeline** with four main stages:

### 1. Synchronized Clip Extraction
- Extracts (frame + audio) pairs at regular time intervals (default: 2 seconds)
- Uses OpenCV for frame capture and FFmpeg for audio segment extraction
- Ensures perfect temporal alignment between visual and audio content

### 2. Multimodal Caption Generation
- **Visual**: BLIP-2 generates natural language descriptions of each frame
- **Audio**: Whisper transcribes speech from audio segments
- **Fusion**: Combines visual and audio captions at each timestamp

### 3. Key Moment Selection
- Intelligently selects ~20 most important moments from all multimodal captions
- Uses uniform sampling (with potential for diversity-based selection)

### 4. Narrative Summary Generation
- Creates timeline of key moments with timestamps and multimodal captions
- Optional LLM-based narrative summarization via OpenRouter API
- Fallback to formatted timeline if LLM unavailable

This **time-synchronized approach** ensures that visual and audio information are always aligned, creating more accurate and contextually rich summaries.
